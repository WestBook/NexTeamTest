# NexTeamTest 專案使用指南

## interactBtn（React + Vite）

- 進入專案目錄並安裝依賴：
  ```bash
  cd interactBtn
  npm install
  ```
- dev 預覽模式：
  ```bash
  npm run dev
  ```

## isCycle（Graph Cycle 判斷腳本）

- 腳本位於 `isCycle/graphCycle.js`，無額外依賴。
- 執行範例：
  ```bash
  cd isCycle
  node graphCycle.js
  ```
- 想測試自訂輸入，可直接修改檔案底部的示範陣列。

## runningClock（React + Vite）

- 進入專案目錄並安裝依賴：
  ```bash
  cd runningClock
  npm install
  ```
- dev 預覽模式：
  ```bash
  npm run dev
  ```
